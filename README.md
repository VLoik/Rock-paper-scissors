# Rock-paper-scissors
Пример клиент-серверного приложения на языке Python, реализующий игру "Камень ножницы бумага".  
Клиент написан с использованием PyQt5.  
Сервер написан на чистом Python и использует сокеты.  
Клиент отправляет на сервер данные о выборе пользователя, сервер обрабатывает его и отсылает результат игры: Победа, поражение или ничья.  
Обмен сообщениями происходит по протоколу TCP/IP
